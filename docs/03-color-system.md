# 03-color-system.md

## Primary Palette
- `#1E40AF` – Primary Blue (CTA buttons)
- `#F1F5F9` – Light Background
- `#111827` – Headings
- `#6B7280` – Secondary Text

## Accent Colors
- `#10B981` – Success / Ready State
- `#F59E0B` – Warning
- `#EF4444` – Error

## Rules
- Stick to these colors for UI consistency
- Use hover states with 10–20% darker versions
