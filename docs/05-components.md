# 05-components.md

## Key Components

### Text Input Form
- Field for user to type content idea
- Button to trigger generation workflow

### Image Display Card
- Preview of AI-generated image
- Download or re-generate option

### Audio Player
- Play/Pause button
- Shows loading state

### Video Composer
- Combines audio and image
- Shows preview + download

### Notification Toasts
- Success / error messages
